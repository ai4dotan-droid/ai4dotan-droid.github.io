  <h2>מהי תוכנית פסגות?</h2>
  <p>
    <span class="highlight">תוכנית פסגות</span> היא תוכנית מצוינות יוקרתית של העתודה האקדמית, שמטרתה להכשיר בוגרי תיכון מצטיינים לתפקידי מחקר ופיתוח בצה״ל ובמערכת הביטחון.
    משתתפי התוכנית מתחילים עם לימודים אקדמיים ארבע-שנתיים, שבמהלכם יש דחיית שירות. בחופשות עוברים החניכים סדרות הכשרה מקצועיות, חינוכיות וצבאיות.
    בסיום הלימודים, ממשיכים החניכים לשירות חובה (32 חודשים) ולאחר מכן שירות קבע (36 חודשים), תוך שילוב קורס קצינים.
  </p>
